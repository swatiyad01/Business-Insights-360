# 📊 Business Insights 360

## 📌 Project Overview
**Business Insights 360** is a comprehensive Power BI dashboard designed to analyze and visualize key performance metrics across multiple departments at **AtliQ Hardware**.

The dashboard delivers actionable insights across finance, sales, marketing, supply chain, and executive operations—enabling stakeholders to make data-driven decisions for strategic growth.

---

## 🚀 Features

### 💰 Finance View
- Profit & Loss (P&L) analysis  
- Product-level financial insights  
- Net sales trends  

### 📈 Sales View
- Customer performance analysis  
- Product performance tracking  
- Interactive charts with dynamic filters  

### 📊 Marketing View
- Regional performance insights  
- Product-level marketing metrics  

### 🚚 Supply Chain View
- Forecast accuracy analysis  
- Net error tracking  

### 🧠 Executive View
- High-level KPI dashboards  
- Revenue breakdowns for quick decision-making  

---

## 🗂️ Data Sources

The dashboard integrates data from multiple sources:

- **Excel/CSV Files**
  - Targets data  
  - Market share data  
  - Supporting datasets  

- **MySQL Database**
  - Transactional and operational data
